\begin{frame}{Assessing parallel trends}

\begin{itemize}
\item The parallel trends assumption is that the treated group in the treatment period would have had the same trend as the control group
\item It is most credible (but not guaranteed) when the two groups previously had similar trends
\end{itemize}
